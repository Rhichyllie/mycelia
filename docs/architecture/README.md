# Architecture Alignment Scaffolding

This folder contains implementation alignment artifacts for MYCELIA.

These files help future tasks map canonical documents to planned modules,
registries, and implementation status. They are not canonical architecture and
must not override `docs/mycelia/`.

Canonical architecture remains in `docs/mycelia/`. If an implementation decision
is not covered by canonical documents, use the ADR process in `docs/adrs/`
instead of inventing behavior.
