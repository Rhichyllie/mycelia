# Contract Registries

This folder contains placeholder roots for future MYCELIA contract registries.
There are no active runtime contracts, schemas, generated clients, generated
servers, migrations, API endpoints, or tool definitions here yet.

Future contract files must align with the owning documents:

- Document 07 for events and event schema governance;
- Document 10 for memory and context contracts;
- Document 11 for policy and approval contracts;
- Document 12 for telemetry semantic conventions;
- Document 13 for security events and evidence contracts;
- Document 14 for tenant boundary requirements;
- Document 15 for tool runtime contracts;
- Document 18 for external API and integration contracts;
- Document 19 for Codex implementation governance.

Do not add contracts without an owner document reference. Do not invent event,
API, tool, schema, or integration names outside canonical documents or an
approved ADR.
