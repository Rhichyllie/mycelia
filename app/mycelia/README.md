# MYCELIA Product Hub Route

The `/mycelia` route is a static product information hub for MYCELIA.

It links to `/mycelia/static-demo`, `/mycelia/walkthrough`, `/mycelia/roadmap`, and `/mycelia/executive`, and explains the current descriptor-level product surfaces, the static demo boundary, and the capabilities that are intentionally not implemented yet.

The hub uses the shared product surface index to present the current route catalog consistently with the home surface and product shell.

This route is static, read-only and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
