# MYCELIA Roadmap Route

The `/mycelia/roadmap` route is a static product roadmap surface for MYCELIA.

It links to `/mycelia`, `/mycelia/static-demo`, `/mycelia/walkthrough`, and `/mycelia/executive`, and explains available static surfaces, planned product direction, inactive capabilities and safety boundaries.

This route is static, read-only and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
