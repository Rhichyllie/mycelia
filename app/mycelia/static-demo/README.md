# MYCELIA Static Demo Route

This is the first tracked MYCELIA App Router static demo route.

It renders the existing read-only static demo view from `src/mycelia/read-only-static-demo-page/`. The route is descriptor-level only and presents a static, safe preview of the first MYCELIA demo descriptor chain.

It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
