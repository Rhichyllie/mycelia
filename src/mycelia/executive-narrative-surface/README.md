# Executive Narrative Surface

`/mycelia/executive` is a static executive narrative surface for MYCELIA.

It links to `/mycelia`, `/mycelia/static-demo`, `/mycelia/walkthrough`, and `/mycelia/roadmap`. It is static, read-only and descriptor-level.

It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
