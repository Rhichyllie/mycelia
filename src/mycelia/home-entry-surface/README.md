# Home Entry Surface

This module contains the reusable MYCELIA home entry surface rendered by the App Router `/` route.

It introduces MYCELIA as governed operational intelligence and a governed agentic runtime, then uses the shared product surface index to point to the current descriptor-level routes.

The surface is static, read-only and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
