# Product Roadmap Surface

This module renders the static MYCELIA product roadmap surface used by `/mycelia/roadmap`.

The roadmap explains what exists now, what remains static and descriptor-level, what surfaces are available, what capabilities are planned next, what remains intentionally inactive, and which safety boundaries are enforced.

It links internally to `/mycelia`, `/mycelia/static-demo` and `/mycelia/walkthrough`. It is static, read-only and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
