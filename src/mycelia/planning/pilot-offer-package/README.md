# Pilot Offer Package

This module defines the first MYCELIA pilot offer planning package for the governed compliance/document review flow.

It is static and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.

The module exists to guide founder-led sales, discovery and the next runtime planning phases. It defines the assessment offer, pilot offer, qualification fields, discovery questions, go/no-go criteria, success metrics, offer boundaries and next technical implications.

The BRL price ranges are internal planning assumptions only. They are not guaranteed pricing and should not be presented as buyer-facing commitments.

The Phase 2Q runtime slice technical plan derives its first technical slice from this pilot package, while still remaining non-executing and non-persistent.
