# Static Demo Walkthrough Surface

`/mycelia/walkthrough` is a static guided walkthrough surface for the first MYCELIA demo descriptor chain.

It links to `/mycelia`, `/mycelia/static-demo`, and `/mycelia/roadmap`. It is static, read-only and descriptor-level.

It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
