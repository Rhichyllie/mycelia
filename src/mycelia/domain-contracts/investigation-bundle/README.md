# MYCELIA InvestigationBundle Contract Skeleton

This module defines the Phase 1P InvestigationBundle contract skeleton.

It is a pure TypeScript descriptor contract for tenant-scoped investigation
bundles that group audit timelines, audit references, runtime references,
transition references, correlation context, and safe investigation metadata.

It does not query storage, persist, replay, render UI, export reports, create
downloadable artifacts, call external services, or implement case management.

The module exists only to define investigation bundle descriptors for future
replay and investigation features.
