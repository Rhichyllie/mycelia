# Product Information Surface

This module renders the static MYCELIA product information hub used by `/mycelia`.

The hub explains what MYCELIA is, what current static product surfaces prove, how the static demo fits into the architecture, why descriptor-level boundaries matter, and what remains intentionally unimplemented.

It links internally to `/mycelia/static-demo`. It is static, read-only and descriptor-level. It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
