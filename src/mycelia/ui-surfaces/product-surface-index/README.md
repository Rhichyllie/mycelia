# Product Surface Index

The product surface index is a static, read-only and descriptor-level catalog of current MYCELIA product surfaces.

It provides internal navigation between `/`, `/mycelia`, `/mycelia/executive`, `/mycelia/static-demo`, `/mycelia/walkthrough`, and `/mycelia/roadmap`.

It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
