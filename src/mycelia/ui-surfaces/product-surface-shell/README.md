# Product Surface Shell

This module provides the shared MYCELIA product surface shell for the current App Router pages.

The shell is static, read-only and descriptor-level. It follows the shared product surface index for internal navigation between `/`, `/mycelia`, `/mycelia/executive`, `/mycelia/static-demo`, `/mycelia/walkthrough` and `/mycelia/roadmap`, safety/status badges, and a short product preview note.

It does not execute runtime work, persist data, call APIs, call tools, call external services, export files, generate PDFs or create downloadable artifacts.
